## ✨ Features

### 1. Water Curve
- Enter 4 points (flow, head) + optional **shut‑off** point (Q=0).
- Three input modes: H‑Q only, With BHP, With efficiency.
- Continuous curve via second‑order polynomial.
- Displays H‑Q, BHP, efficiency (%) and marks the Best Efficiency Point (BEP) with a star.
- Export to Excel (multiple sheets) and HTML (interactive plot).

### 2. Affinity Laws
- Scale pump by diameter ratio.
- Generates new curve with head, BHP, efficiency.
- Comparative graph with single legend entry for BEP.
- Export new curve to Excel.

### 3. Viscosity Correction (HI 9.6.7)
- Input: speed (rpm), viscosity (cSt), specific gravity.
- BEP selection: Auto (from generated curve) or Manual (from original 4 points).
- Table with reference points (60%, 80%, 100%, 120% of Qbep) and power increase.
- Full viscosity curve over all flow values.
- Exports table and full curve to Excel.
